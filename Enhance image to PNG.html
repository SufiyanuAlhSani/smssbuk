In HTML, you insert an image using the <img> tag. Here’s the basic syntax:

html
<img src="image.jpg" alt="Description of image" width="300" height="200">

Explanation:

  • src → the path (URL or file location) of the image.

  • alt → alternative text that shows if the image can’t load (important for accessibility and SEO).

  • width & height → optional attributes to set the image size (in pixels).

Example 1: Image from the same folder

html
<img src="logo.png" alt="School Logo" width="200">

Example 2: Image in a subfolder

html
<img src="images/logo.png" alt="School Logo">

Example 3: Image from an external website

html
<img src="https://example.com/logo.png" alt="External Logo">

Example 4: Making image responsive with CSS

html
<img src="logo.png" alt="Responsive Logo" style="max-width:100%; height:auto;">

👉 Place the image file (e.g., logo.png) in the same directory as your HTML file, or adjust the src path accordingly.
